PASSWORD_BCRYPT

PASSWORD_BCRYPT

PHPUnit Process isolation caches constants, but not function declarations.

So we need to check if the constants are defined separately from the functions to enable supporting process isolation in userland code.

password_needs_rehash()

password_needs_rehash(string  $hash, integer  $algo, array  $options = array()) : boolean

Determine if the password hash needs to be rehashed according to the options provided

If the answer is true, after validating the password using password_verify, rehash it.

Parameters

string $hash

The hash to test
integer $algo

The algorithm used for new password hashes
array $options

The options array passed to password_hash

Returns

boolean —

True if the password needs to be rehashed.
